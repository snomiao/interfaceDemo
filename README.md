# interfaceDemo

一个 python post 接口小demo

## 使用说明
在本地运行的话，浏览器访问 http://localhost:8000 就可以看到一个页面，从这个页面可以调用post接口

## 接口说明：
调用方法： POST /
参数（BODY）：

|param|optional|type|comment|
|:-:|:-:|:-:|:-:| 
|firstname|no|string|名字|
|lastname|no|string|姓|
|age|yes|number|年龄|
|height|yes|number|身高|
|weight|yes|number|体重|
